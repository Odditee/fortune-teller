# 需求规格说明

## 项目名称
塔罗占卜 Web 应用 — Stellar Oracle

## 目标用户
个人用户，MacBook Air，不懂代码

## 功能需求

### F1. 牌阵选择
- 10种经典牌阵，中英双语
- 卡片式展示，点击选择

### F2. 抽牌（两种方式）
- 方式A：输入1-78数字，抽牌动画
- 方式B：牌圈交互（Three.js 3D牌圈，鼠标旋转，点击选牌）

### F3. 牌面展示
- 几何花窗风牌面，翻转动画
- 正位/逆位判定

### F4. 内容系统
- 78张牌预置内容库（正逆位含义、象征、典故、案例）

### F5. AI解读
- 混合方案：预置内容 + Claude AI综合解读
- 月费约¥3-8

### F6. 历史记录
- IndexedDB存储，可查看、添加备注

### F7. UI主题
- 星空背景 + 银色冷光
- 欧式古典衬线字体（Cinzel, Cormorant Garamond）
- 中英双语界面
