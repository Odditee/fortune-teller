# UI 设计规范

## 色彩系统

| 变量 | 色值 | 用途 |
|------|------|------|
| --bg-deep | #080c18 | 最深背景色 |
| --bg-night | #0b1025 | 页面/卡片背景 |
| --bg-card | #0e1430 | 卡片底色 |
| --text-primary | #c8d0d8 | 主文字（银白） |
| --text-accent | #b8c5d6 | 强调文字（亮银） |
| --text-dim | #6a7090 | 次要文字（暗银灰） |
| --border-accent | #5a6080 | 边框强调 |
| --accent | #8a90b0 | 主色调（银蓝） |
| --accent-glow | #a0afca | 发光效果 |

## 字体层级

| 用途 | 字体栈 | 字重 |
|------|--------|------|
| 标题 | 'Cinzel', 'EB Garamond', serif | 400-600 |
| 正文 | 'Cormorant Garamond', 'EB Garamond', serif | 300-400 |
| 中文 | 继承上述字体栈，回退系统衬线体 | — |

## 间距系统
- 基础单位: 4px
- 内边距: 8px / 12px / 16px / 24px
- 卡片间距: 12px / 16px
- 圆角: 8px (卡片), 12px (弹窗), 4px (按钮)

## 动画参数

| 动画 | 时长 | 缓动 |
|------|------|------|
| 页面切换 | 300ms | ease-out (opacity) |
| 卡牌翻转 | 600ms | ease-in-out |
| 按钮悬停 | 200ms | ease |
| Toast | 300ms in / 500ms out | ease |
| 星空粒子 | 60fps requestAnimationFrame | — |
| 牌圈旋转惯性 | 0.95 衰减系数 | — |

## 组件规范

### 按钮
- 主按钮: 银蓝边框 + 暗色背景，悬停发光
- Ghost按钮: 透明背景 + 银色文字，悬停下划线
- 模式选择: 卡片式，选中态银色边框发光

### 卡片
- 牌阵卡片: 160px 最小宽度，银色细边框，悬停微上浮
- 牌面卡片: 几何花纹背景，花窗玻璃风格纹理

### 弹窗
- 居中显示，背景模糊遮罩
- 最大宽度 480px
- 右上角关闭按钮

### 牌圈3D
- 78张牌均匀分布在半径3.5的圆环
- 牌面尺寸 0.22×0.34 (3D单位)
- 选中状态: emissive 银色发光
- 惯性旋转衰减 0.95/帧
